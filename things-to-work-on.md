## Slicing Lists II

Example:

```
fruits = ["apple", "cherry", "pineapple", "orange", "mango"]
```

To select the first *x* number of elements in the list: `fruits[:x]`\
(Starting from the index number 0)

To select the last *x* number of elements in the list: `fruits[x:]`
